**Neurotech Lie Detector Prototype**

This project is a prototype application built to explore real-time neurotechnology data visualization using Neuropawn EEG hardware.

**Overview**

This was a 1-month school project where I was given early access to new Neuropawn headsets. Since official software was not functioning properly on school systems and documentation was limited, I had to work directly with raw device data and develop my own tools to interface with it.

The main focus of the project became building a real-time data visualization system for the headset, due to hardware and setup limitations.

**What I Built**

Real-time data visualization pipeline for Neuropawn EEG data
Lie detection demo application using user input and signal response patterns
Interface for displaying streaming sensor data

**How It Works**

Users are prompted with questions and asked to type responses while minimizing movement. The system visualizes real-time signal changes from the headset during input.

**Challenges**
Limited documents for Neuropawn hardware
Visualization software did not work in the school setting
Time-consuming process of data synchronization and hardware troubleshooting
Raw signals had to be interpreted by trial and error

**Outcome**

Due to hardware constraints, most development time was spent on:

Building a stable real-time visualization system
Understanding and processing incoming sensor data

The lie detection component is a basic prototype built on top of this system.

**What I Would Improve**

If revisiting this project, I would:

Focus on hardware troubleshooting earlier
Allocate more time to signal processing and model development
Improve data interpretation methods for more meaningful outputs
